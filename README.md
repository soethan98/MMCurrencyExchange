# MMCurrencyExchange

 Display exchange rate fetched from https://forex.cbm.gov.mm/api/
 
 # Components
 - Android Architecture components(Room,livedata,viewmodel)
 - Coroutines
 - Timber
 - Retrofit
 - Dependency Injection(Koin)
